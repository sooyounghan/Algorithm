### 확장 유클리드 호제법
-----
1. 유클리드 호제법의 목적이 두 수의 최대 공약수를 구하는 것이라면 확장 유클리드 호제법의 목적은 방정식의 해를 구하는 것
2. 해를 구하고자 하는 방정식 : ax + by = c (a, b, c, x, y는 정수)
   - 위 방정식은 c % gcd(a, b) = 0인 경우에만 정수해를 가짐
   - 다시 말해, c가 a, b의 최대 공약수의 배수인 경우에만 정수해를 가짐 : 이는 ax + by = c가 정수해를 갖게 하는 c의 최솟값이 gcd(a, b)라는 것을 의미
   - 구현에는 재귀 함수 사용

3. 확장 유클리드 호제법의 원리 이해하기 (예) 5x + 9y = 2일 때, 이 식을 만족하는 정수 x, y를 구하기)
   - 우선 5x + 9y가 정수해를 갖게 하는 c의 최솟값이 gcd(5, 9) : gcd(5, 9) = 1이므로 5x + 9y = 1로 식을 다시 놓고 진행
   - a, b로 유클리드 호제법을 반복 실행하며 몫, 나머지를 저장 : 반복은 나머지가 0이 되면 중단

   - 반복으로 구한 나머지와 몫을 이용해 거꾸로 올라가며 x = y', y = x' - y' * q를 계산
     + x'는 이전 x, y'는 이전 y를 의미, q는 현재 보고 있는 몫을 의미
     + 이 때, 처음 시작하는 x, y는 이전 x와 y가 없으므로 각각 1, 0으로 지정하여 역 계산 진행

   - 이렇게 재귀 방식으로 알아낸 최종 x, y는 ax + by = gcd(a, b)를 만족
     + 그리고 c / gcd(a, b) = K라고 가정하면, 최초 방정식의 해는 Kx, Ky로 간단히 구할 수 있음
     + 3번쨰 과정에서 찾은 x는 2, y는 -1이므로 2 / 1 = 2이며, 2 * 2, 2 * -1에 의해 최초 방정식의 해는 4, -2

4. 오른쪽 변의 값이 gcd(a, b)의 배수가 아닐 경우
   - 결론적으로 이 경우를 만족하는 x, y 값은 정수 범위에서 존재하지 않음
   - 따라서 유클리드 호제법을 구현할 때, 먼저 오른쪽 변의 값이 gcd(a, b)의 배수라는 조건을 만족하는지 먼저 판단해야 함
   - 만약 조건을 만족하지 않는다면, 이후 프로그램을 수행하지 않고 불가능을 표현하는 값을 출력하면 됨
   - 또한, 유클리드 호제법 구조 자체가 특정한 값을 업데이트시키면서 같은 로직을 반복적으로 수행하므로 재귀 함수 형태로 구현
