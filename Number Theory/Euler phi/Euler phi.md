### 오일러 피
-----
1. 오일러 피 함수 P[N]의 정의는 1부터 N까지 범위에서 N과 서로소인 자연수의 개수를 뜻하
2. 오일러 피 함수의 원리는 에라토스테네스의 체와 비슷
   - 구하고자 하는 오일러 피의 범위만큼 배열 초기화
   - 2부터 시작해 현재 배열의 값과 인덱스가 같으면(= 소수일 때) 현재 선택된 숫자(K)의 배수에 해당하는 수를 배열 끝까지 탐색하여 P[i] = P[i] - P[i] / k 연산을 수행 (i는 K의 배수)
   - 배열의 끝까지 2번째 과정을 반복해 오일러 피 함수를 완성

4. 수학적으로 오일러 피 함수 이해
   - 초기 상태 : φ(6) = 6 (서로소가 될 수 있는 후보로 개수 초기화(1, 2, 3, 4, 5, 6))
   - 2의 배수로 인한 후보 탈락 : φ(6) = 6 - (6 / 2) = 3 (1, 3, 5)
   - 3의 배수로 인한 후보 탈락 : φ(6) = 3 - (3 / 2) = 2 (1, 5)
   - 이 떄 후보에서 삭제하는 기준을 6이 아닌 업데이트 된 3으로 진행하는 이유는 3의 배수 중 2인 배수인 수들은, 즉, 3과 2의 공배수는 2의 배수에서 이미 삭제되었으므로 중복 삭제를 막기 위함
   - 최종적으로 φ(6) = 2가 되며, 이 때 2의 의미는 6이하 숫자들 중 서로소가 되는 개수가 2개 (1, 5)라는 뜻

5. 오일러 피 문제는 출제되는 빈도가 많지 않지만, 원리를 알지 못하면 출제됐을 때 문제 자체가 접근이 어려워짐
