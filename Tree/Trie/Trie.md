-----
### 트라이
-----
1. 문자열 검색을 빠르게 실행할 수 있도록 설계한 트리 형태 자료구조
2. 일반적으로 단어들을 사전 형태로 생성한 후 트리의 부모-자식 노드를 관계를 이용해 검색 수행
3. 트라이 특징
   - N진 트리 : 문자 종류의 개수에 따라 N이 결정 (예를 들어, 알파벳은 26개 문자로 이뤄져있으므로 26진 트리로 구성
   - 루트 노드는 항상 빈 문자열을 뜻하는 공백 상태 유지

4. 영단어 apple, air, apply를 순서대로 트라이 자료구조에 삽입

   - 루트 노드는 공백을 유지하고 apple의 각 알파벳에 해당하는 노드를 생성
   - 다음으로 air를 삽입할 때는 루트 노드에서부터 검색 : a 노드는 공백 상태가 아니므로 이동하고, i와 r은 공백 상태이므로 신규 노드를 생성
   - apply를 삽입할 때도 검색 노드가 공백 상태이면 신규 노드를 생성하고, 아니면 이동하는 원리로 트라이 자료구조 구현
