### 기수 정렬
-----
1. 값을 비교하지 않는 특이한 정렬
2. 값을 놓고 비교할 자릿수를 정한 다음, 해당 자릿수만 비교
3. 시간 복잡도는 O(kn)으로, 여기서 k는 데이터 자릿수 (예를 들어, 234, 123을 비교히면 4와 3, 3과 2, 2와 1만 비교)
4. 기수 정렬은 10개의 큐를 이용 : 각 큐는 값의 자릿수를 대표

5. 수행 방식
   - 원본 배열은 16, 80, 18, 77, 03, 24, 88, 23
   - 먼저 일의 자릿수 기준으로 배열 원소를 큐에 집어넣음
     + 그런 다음 0번째 큐부터 9번쨰 큐까지 pop을 진행 : 결과 80, 03, 23, 24, 16, 77, 18, 88이 만들어짐
   - 이어서 십의 자릿수를 기준으로 같은 과정 진행
   - 마지막 자릿수를 기준으로 정렬할 때까지 앞의 과정 반복

6. 기수 정렬은 시간 복잡도가 가장 짧은 정렬 : 정렬해야 하는 데이터가 너무 많으면 활용하는 것이 좋음
