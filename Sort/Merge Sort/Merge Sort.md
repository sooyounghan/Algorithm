### 병합 정렬
-----
1. 분할 정복(Divide and Conquer) 방식을 사용해 데이터를 분할하고 분할한 집합을 정렬하며 합치는 알고리즘
2. 시간 복잡도 평균값은 O(n log n)
3. 병합 정렬의 이해 : 집합은 setN으로 표시

4. 수행 방식
   - 처음에는 8개 그룹으로 나누며, 이 상태에서 2개씩 그룹을 합치며 오름차순 정렬 : 결과 (32, 42), (24, 60), (5, 15), (45, 90)
   - 이어서 2개씩 그룹을 합치며 다시 오름차순 정렬 : 결과 (24, 32, 42, 60), (5, 15, 45, 90)
   - 이런 방식으로 병합 정렬 과정을 거치면 (5, 15, 24, 32, 42, 45, 60, 90)이 되어 전체를 오름차순 정렬 가능

5. 코딩 테스트 정렬 문제로 자주 등장 : 특히 2개의 그룹을 병합하는 원리를 꼭 숙지

6. 2개의 그룹을 병합하는 과정
   - 투 포인터 개념을 사용하여 왼쪽, 오른쪽 그룹을 병합
   - 왼쪽 포인터와 오른쪽 포인터의 값을 비교하여 작은 값을 결과 배열에 추가하고, 포인터를 오른쪽으로 1칸 이동
