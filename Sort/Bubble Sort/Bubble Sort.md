### 버블 정렬 (Bubble Sort)
-----
1. 두 인접한 데이터의 크기를 비교해 정렬하는 방법
2. 간단하게 구현할 수 있지만, 시간 복잡도는 O($n^{2}$)으로 다른 정렬 알고리즘보다 속도가 느린편
3. 루프(Loop)를 돌면서 인접한 데이터 간의 swap 연산으로 정렬
<div align="center">
<img src="https://github.com/user-attachments/assets/05c14ae5-e56b-442f-9e2b-35ec3f391cc6">
</div>

4. 정렬 과정
   - 비교 연산이 필요한 루프 범위를 설정
   - 인접한 데이터 값을 비교
   - swap 조건에 부합하면 swap 연산 수행
   - 루프 범위가 끝날 때까지 2 ~ 3번째 단계 반복
   - 정렬된 영역을 설정하고, 다음 루프를 실행할 때는 이 영역을 제외
   - 비교 대상이 없을 때까지 1 ~ 5번째 단계 반복

5. 만약 특정한 루프의 전체 영역에서 swap이 한 번도 발생하지 않았다면, 그 영역 뒤에 있는 데이터가 모두 정렬됐다는 뜻으로 프로세스를 종료해도 됨
