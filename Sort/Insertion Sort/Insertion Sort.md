-----
### 삽입 정렬 (Insertion Sort)
-----
1. 이미 정렬된 데이터 범위에 정렬되지 않은 데이터를 적절한 위치에 삽입시켜 정렬하는 방법
2. 평균 시간 복잡도는 O($n^{2}$)으로 느린 편이지만, 구현하기가 쉬움
3. 선택 데이터를 현재 정렬된 데이터 범위 내 적절합 위치에 삽입하는 것이 핵심

4. 삽입 정렬 과정
   - 현재 index에 있는 데이터 값을 선택
   - 현재 선택한 데이터가 정렬된 데이터 범위에 삽입될 위치를 탐색
   - 삽입 위치부터 index에 있는 위치까지 shift 연산 수행
   - 삽입 위치에 현재 선택한 데이터를 삽입하고 index++ 연산 수행
   - 전체 데이터 크기만큼 index가 커질 때까지, 즉 선택할 데이터가 없을 떄까지 반복

5. 적절한 삽입 위치를 탐색하는 부분에서 이진 탐색(Binary Search) 등과 같은 탐색 알고리즘을 사용하면 시간 복잡도를 줄일 수 있음
