-----
### 이진 탐색
-----
1. 데이터가 정렬되어있는 상태에서 원하는 값을 찾아내는 알고리즘
2. 대상 데이터의 중앙값과 찾고자 하는 값을 비교해 데이터 크기를 절반씩 줄이면서 대상을 찾음

3. 이진 탐색은 정렬 데이터에서 원하는 데이터를 탐색할 때 사용하는 가장 일반적인 알고리즘
4. 구현 및 원리가 비교적 간단하므로 많은 코딩 테스트에서 부분 문제로 요구하는 영역
5. 이진 탐색은 오름차순으로 정렬된 데이터에서 다음 4가지 과정 반복 (내림차순의 경우 조건을 반대로 하여 과정 반복)
   - 현재 데이터셋의 중앙값(Median)을 선택
   - 중앙값 > 타겟 데이터(Target Data)일 때, 중앙값 기준으로 왼쪽 데이터셋 선택
   - 중앙갑 < 타겟 데이터일 때, 중앙값 기준 오른쪽 데이터셋 선택
   - 세 과정을 반복하다가 중앙값 == 타깃 데이터일 때 탐색 종료

6. 예) 총 16개의 데이터가 있는 데이터셋에서 값이 55인 데이터를 찾는 과정

7. 이진 탐색을 사용하면 N개의 데이터에서 log(N)번의 연산으로 원하는 데이터의 위치를 찾을 수 있음
   - 예를 들어 16개의 데이터면 최대 4번의 연산으로 원하는 데이터 위치를 찾을 수 있음
   - 다만, 이진 탐색은 데이터가 정렬되어 있어야 함
